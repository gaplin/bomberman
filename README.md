# BOMBERMAN
![bombermanGif](https://user-images.githubusercontent.com/50521366/218258786-b8fec21e-0369-4730-89dc-839ab546279b.gif)

## Requirements 
* java 8 or higher
## Menu controls
* Arrow keys + Enter
## Game Controls
* P to pause/unpause
* Player 1
    * Arrow keys to move
    * K to place bomb
* Player 2
    * W/S/A/D to move
    * T to place Bomb
## Other Controls
* M to mute/unmute
* F to show/hide FPS counter
